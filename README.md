# 🚓 Police Data Analysis Project

## 📌 Overview
This project analyzes a police dataset using Python and Pandas to extract meaningful insights and patterns.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## 📊 Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values
- Filtering and grouping data
- Aggregations and insights
- Visualization of patterns

## 📁 Files
- Police Dataset.ipynb → Main analysis notebook

## 🚀 How to Run
1. Clone the repository
2. Open the notebook in Jupyter Notebook / VS Code
3. Run all cells

## 💡 Skills Demonstrated
- Data Analysis
- Problem Solving (DSA thinking)
- Data Cleaning
- Visualization

## 📌 Conclusion
This project demonstrates how raw data can be transformed into meaningful insights using Python.
